⚙ Quick reference — Modal verbs (B2)

FunctionModalsExample
Obligation / necessitymust, have to, need toYou must enable 2FA.
Prohibitionmustn't, can'tYou mustn't share your password.
No obligationdon't have to, needn'tYou don't have to use the VPN here.
Adviceshould, ought to, had betterYou should back up your data.
Deduction (now)must be / can't be / might beThe server must be down.
Deduction (past)must have / can't have / should have + doneSomeone must have deployed broken code.
Ability / permissioncan, could, be able to, be allowed toOnly admins can access the root account.
